Controls Overview

Following is a list of controls supported by Synchro.  

For a common attributes supported by all controls, see: Common Control Attributes.

Otherwise select a control below to see the additional attributes that it supports, as well as data binding support and samples. 

Border Control

The border control is a container that holds a single control and displays a visible border around it.

Button Control

A standard button control.

Canvas Control

The canvas control is a container that holds one or more controls, which can be positioned relative to the canvas itself (most notably, controls in a canvas container can be made to overlap).

Edit Control

A standard edit control.

Image Control

A control to display an image resource.

Listbox Control

A control to display, and potentially allow interaction with, a list of text items.

ListView Control

A control to display, and potentially allow interaction with, a list of view items (items that are themselves composed of one or more controls).

Location Control

Non-visible (non-layout) control for providing location services.

Password Control

A standard edit control that does not disclose values entered into the control.

Picker Control

A control to let the user pick from a list of items.

ProgressBar Control

The ProgressBar control shows percent completion of a task.

ProgressRing Control

The ProgressRing control indicates that an activity is occurring (often implying that the user is waiting for completion).

Rectangle Control

A simple rectangle control, which can have a border and be filled as desired.

ScrollView Control

A container that holds a single control and scrolls that control within the area of the scroll control.

Slider Control

A slider control that can be dragged through a range of positions representing a range of values.

StackPanel Control

A container control for organizing a set of container controls either horizontally or vertically.

Text Control

A title or label control for displaying text.

Toggle Control

An on/off toggle control.

WebView Control

A control that displays a web view, populated either with local content, or loaded from a URL.

WrapPanel Control

A container control that flows contents either vertically or horizontally, and wraps overflowing content onto the next row/column.

Android: ActionBar Controls

An Android ActionBar controls.

iOS: NavBar and ToolBar Controls

iOS NavBar and ToolBar controls.

Windows: CommandBar Controls

Windows CommandBar controls.
